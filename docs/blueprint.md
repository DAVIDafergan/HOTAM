# **App Name**: Ketav Connection

## Core Features:

- Seller Onboarding: Multi-step registration process for sellers, including personal information, qualifications (certificate, Torah study, mikveh), and script samples.
- Customer Registration: Registration process for customers via email/phone or Google/Apple.
- Precise Product Search: Detailed search functionality with filters for product type (Mezuzah, Tefillin, Megillah, Sefer Torah, etc.), script type (Sephardi, Ashkenazi), level of script quality, and additional filters (parchment size, scribe qualifications, etc.).
- Product Display: Display of products matching search criteria, with sorting by price and location.  Message displayed when no products match, with option to request a custom order.
- Product Upload: Interface for sellers to upload product details, including type, quantity, script, level of script quality, parchment size, and level of proofreading.
- Seller Profile: Display detailed information about the seller, including photos, qualifications, city, etc. along with a review system.
- AI-Powered Mezuzah Diagnosis: Customers can upload pictures of Mezuzah scrolls that will be assessed by an AI tool.

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to convey trust and tradition.
- Background color: Very light grayish-blue (#F0F4F7), for a clean and calming backdrop.
- Accent color: Warm Gold (#FFC107) to highlight important elements.
- Body and headline font: 'Alegreya', a serif font to convey tradition and readability.
- Use of symbolic and meaningful icons, keeping a consistent style throughout the platform.
- Clean and structured layout to facilitate easy navigation and browsing. Use of grid system to organize products and information effectively.
- Subtle transitions and loading animations to provide a smooth user experience.
